# 东南大学生医工竞赛


## —基于脑电数据的睡眠质量分析研究

##### ！在运行本仓库的代码前请确保已完成Pycharm及conda环境的配置


### 文件结构说明
##### 1./code 存放项目核心代码文件
##### 2./data 包含测试数据（/fake_data中的数据是随机生成以供测试的，不是竞赛方提供）
##### 3./resources收集本次比赛的题目，报名表，指导等文件
##### 4./results放置最终生成的结果
##### 5./report存放最终的报告

### 实现路径分析


##### 1.数据读取与信号预处理
##### 2.脑电信号滤波
##### 3.30s帧分割与标签匹配
##### 4.脑电特征提取
##### 5.简单模型训练与验证
##### 6.测试集预测和结果文件生成
##### 7.报告撰写
