### 文件结构说明
##### 1./code 存放项目核心代码文件
##### 2./data 包含测试数据
- /fake_data中的数据是随机生成以供测试的，内含的.exe可以直接生成少量假数据（已不兼容）
- /train_data存放有所有训练集数据
- /processed_data存放预处理后的数据及模型
##### 3./resources收集本次比赛的题目，报名表，指导等文件
##### 4./results放置最终生成的结果
##### 5./report存放最终的报告

### 主程序运行结构
##### 1.data_preprocessing.py
```graph TD
  A[遍历并区分数据和标签] --> B[按照ID匹配数据和标签]
  B --> C[读取并筛选出有效标签]
  C --> D[验证结果确保对应]
  D --> E[进行帧分割/滤波]
  E --> F[数据保存为npy文件]
```
##### 2.model_training.py
```graph TD
  A[读取npy文件] --> B[提取时域+频域特征]
  B --> C[标签编码（R/1/2/3→数字）]
  C --> D[划分训练/测试集]
  D --> E[训练随机森林模型]
  E --> F[评估宏F1值]
  F --> G[保存模型/特征]
```
